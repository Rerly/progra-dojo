中級：DOM操作の基本

JavaScriptを使って、Webページの内容を動的に変更できるようになります。

学習目標

  • DOMとは何かを説明できる
  • documentの役割を理解する
  • getElementByIdの仕組みを理解する
  • textContentで文字を変更できる

DOMとは？

DOM（Document Object Model）とは、 WebページをJavaScriptから操作できる仕組みのことです。

HTMLで作った要素（h1やpなど）を、 JavaScriptで取得し、変更できるようになります。

基本構造

JavaScript 

function changeText() {
  const element = document.getElementById("text");
  element.textContent = "やった！";
}

構造を分解してみよう

  • function → 動作のまとまりを作る
  • document → Webページ全体を表す
  • getElementById → idで要素を取得する
  • textContent → 要素の文字を変更する

① ボタンで文字を変えてみよう

こんにちは

JavaScript 

const btn1 = document.getElementById("changeBtn");
const text1 = document.getElementById("text1");

btn1.addEventListener("click", function() {
  text1.textContent = "やった！";
});

ここに表示されます

JavaScript 

const input = document.getElementById("liveInput");
const output = document.getElementById("liveOutput");

input.addEventListener("input", function() {
  output.textContent = input.value;
});

色が変わるよ

const btn = document.getElementById("colorBtn");
const text = document.getElementById("colorText");

btn.addEventListener("click", function() {

④ 要素を追加してみよう

    JavaScript 
    
const addBtn = document.getElementById("addBtn");
const list = document.getElementById("listArea");

addBtn.addEventListener("click", function() {
  const li = document.createElement("li");
  li.textContent = "新しい項目";
  list.appendChild(li);
});

    ⑤ イベントオブジェクトとは？

    イベントが発生すると、JavaScriptは自動的に eventオブジェクトを渡してくれる。

    イベントオブジェクトの基本 
    
element.addEventListener("click", function(event) {
  console.log(event);
});
    • event.target → クリックされた要素
    • event.type → イベントの種類
    • event.preventDefault() → デフォルト動作を止める

    ⑥ フォームと preventDefault()

    JavaScript 
    
form.addEventListener("submit", function(event) {
  event.preventDefault();
  message.textContent = "送信は止められました";
});
